Yes. Outgoing calls run pre-call risk checks; incoming calls are screened with live detection and verified caller identity before you answer.

No by default. Scoring runs on-device. Enterprises can enable opt-in telemetry for quality improvements.

How does it detect deepfakes?

It analyzes audio frame and artifacts using ML models tuned for voice cloning patterns.
